# Link Shortener

A modern and responsive Link Shortener web app built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

This project allows users to enter a long URL, generate a shortened version, copy it easily, and manage saved links with a clean user interface.

## Features

- Add long URLs and generate short links
- URL validation
- Copy shortened links to clipboard
- Delete saved links
- Data saved in `localStorage`
- Responsive and modern UI
- Built with TypeScript for better type safety

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
